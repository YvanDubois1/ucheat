  • Version 0.5 : ajout de liens directs vers UHAUL.COM pour chaque article ; mise à jour des affichages flex-box ; ajout d'une note en haut de page concernant les paramètres des articles.
  • Version 0.4 : correction d'un bug d'affichage lorsque le tarif kilométrique du week-end était de 1,09 $.
  • Version 0.3 : correction de quelques fautes de frappe.
  • Version 0.2 : Il s'agit d'une version bêta (test) de la fonctionnalité permettant de définir les paramètres des articles comme les prix et la visibilité. Elle n'a **pas** été testée de manière approfondie et doit être vérifiée attentivement lors de son utilisation. Si vous trouvez des problèmes, veuillez me le faire savoir sur Reddit : /u/uhaulcsr ou si vous êtes familier avec GitHub, vous pouvez ouvrir une issue.
  • Un rappel supplémentaire : les paramètres sont stockés **par appareil** à l'aide d'un cookie. Vous devrez définir les paramètres une fois par appareil, et si vous effacez les cookies, ils seront réinitialisés aux valeurs prédéfinies qui pourraient ne pas correspondre à votre marché.
  • Cette version est assez rudimentaire. Si vous êtes familier avec le codage, n'hésitez pas à participer au développement sur GitHub.
